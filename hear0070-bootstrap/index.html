1. Bootstrap is a user friendly framework for building responsive template starter pages! First start by placing scripts that can be found on the getbootstrap.com to the bottom of your HTML page, or you can simply copy paste the premade starter template to your HTML!
2. Bootstrap uses custom made css classes to apply to your code to save you time and effort! Go ahead and try it out by simply applying a class to one of your attributes!
3. If you're looking to dive even deeper into what Bootstrap has to offer! Visit getbootstrap.com to learn more. There you can find everything you need to know.

A few examples of classes that you can test out are:

  • display-1, display-2 and so on which directly effect fontsizing.

  • text-primary, text-secondary which can change the color theming of your text.

  • The "m" class defines a margin on an attribute, for example: "mt-3" will set a margin to the top of the element.
